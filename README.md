# ❤️ Sudoeste Meu Amor

Plataforma criada para conectar moradores do Sudoeste (Brasília) com empresas locais.

A ideia é criar um ambiente onde empresas possam:

- Criar seu perfil
- Mostrar informações da empresa
- Oferecer descontos exclusivos
- Receber avaliações de clientes
- Aparecer no mapa da região

## 🚀 Objetivo

Fortalecer o comércio local do Sudoeste e facilitar a descoberta de serviços e estabelecimentos da região.

## 📍 Funcionalidades planejadas

- Cadastro de empresas
- Perfil da empresa com:
  - Logo
  - Fotos
  - Descrição
  - Endereço
- Integração com mapa
- Avaliação de usuários
- Ofertas e descontos exclusivos
- Descoberta de lugares no bairro

## 🛠 Tecnologias

Primeira versão construída com:

- HTML
- CSS
- JavaScript

Planejado para evoluir para:

- Backend
- Banco de dados
- Sistema de login
- API de mapas

## 🌎 Projeto

Inspirado no perfil do Instagram **Sudoeste Meu Amor**, focado em valorizar os negócios locais da região.
